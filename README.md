
# 📱 Leecode Mobile Application (Flutter)

A **LeetCode-inspired mobile application** built using **Flutter**, designed to provide a clean and intuitive platform for practicing coding problems directly from a mobile device. The app focuses on delivering a **modern UI, smooth navigation, and a developer-friendly experience** similar to the LeetCode web platform, optimized for mobile learning.

This project was built using **Flutter with Antigravity development workflow**, focusing primarily on **UI/UX design and structured problem browsing for coding practice**.

---

# 🚀 Features

- 📚 **Coding Problem Library** – Browse algorithm and data structure problems.
- 🔍 **Problem Categories** – Organized by difficulty (Easy, Medium, Hard).
- 📱 **Mobile-First UI** – Designed specifically for a smooth mobile experience.
- 🎨 **Modern Flutter UI** – Clean and responsive interface.
- ⚡ **Fast Navigation** – Efficient routing between problems and categories.
- 🧠 **Coding Practice Focused** – Built for competitive programming learners.

# ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/aliiakbarkhan/leecode-application-flutter-antigravity.git
```

### 2️⃣ Navigate to the Project

```bash
cd leecode-application-flutter-antigravity
```

### 3️⃣ Install Dependencies

```bash
flutter pub get
```

### 4️⃣ Run the Application

```bash
flutter run
```
